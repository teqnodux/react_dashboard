# Tearsheet Enhancement Roadmap

## ✅ Completed (Critical - Items 1-4)
- [x] **Sortable columns** - Click headers to sort by spread %, value, date
- [x] **Category grouping** - Collapsible category rows (Low-risk, Higher-risk, LBOs, etc.)
- [x] **Sticky header** - Header stays visible when scrolling
- [x] **Keyboard navigation** - Arrow keys to navigate, Enter to open detail

## 📋 Backlog (Priority Order)

### High Priority (Functionality)
5. **Quick filters dropdown** - Filter by:
   - Deal type (Cash/Stock/Mixed)
   - Spread range (tight <3%, mid 3-8%, wide >8%)
   - Risk level (Low/Medium/High)
   - Close timing (Near-term/Mid-term/Long-term)

6. **Column visibility toggles** - Show/hide entire column groups

7. **Export to Excel/CSV** - Download current filtered view

8. **Bulk actions** - Select multiple deals to add/remove from watchlist

### Medium Priority (Visual Enhancements)
9. **Color coding** - Highlight:
   - Spreads (green tight, yellow mid, red wide)
   - Risk levels (highlight at-risk deals)
   - Recent updates (deals with new activity)

10. **Conditional formatting** - Visual indicators for:
    - Regulatory challenges (red flag icon)
    - Approaching outside date (warning icon)
    - High downside risk (caution icon)

11. **Mini sparklines** - Show spread trend over time

### Advanced Features
12. **Save custom views** - Save filter/sort combinations

13. **Column reordering** - Drag to reorder columns

14. **Refresh indicator** - Show when quote data was last updated per deal

15. **Deal comparison** - Select 2-3 deals to compare side-by-side

## Notes
- Items 1-4 provide immediate value for daily usage
- Items 5-8 add power-user functionality
- Items 9-11 improve visual clarity at-a-glance
- Items 12-15 are advanced features for sophisticated workflows
